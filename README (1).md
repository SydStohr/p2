# 🔗 LinkLogistics — Supply Chain Simulator

A Streamlit-based interactive dashboard for teaching Supply Chain Management.

---

## ▶️ How to run

### Step 1 — Install Python
Make sure Python 3.9 or higher is installed:
https://www.python.org/downloads/

### Step 2 — Install Streamlit
Open your terminal (or Command Prompt on Windows) and run:

```
pip install streamlit
```

### Step 3 — Run the app
Navigate to the folder where you saved `linklogistics_app.py` and run:

```
streamlit run linklogistics_app.py
```

Your browser will automatically open at: http://localhost:8501

---

## 📁 Files
- `linklogistics_app.py` — the main app (this is all you need)
- `requirements.txt` — lists the Python packages needed

---

## 🎮 What's included
- **Dashboard** — Live score, inventory bars, leaderboard, P&L, sustainability breakdown
- **Mini-games** — 4 interactive challenges (Supplier selection, Demand forecasting, Crisis response, Logistics route)
- **Financials** — Income statement and balance sheet
- **Instructor view** — Trigger global events, advance quarters, view all teams

---

## 💡 Tips
- The score updates live as you complete mini-games correctly
- The Instructor can trigger Black Swan events that change the context across the dashboard
- Use "Advance quarter" in Instructor view to reset mini-games for the next round
